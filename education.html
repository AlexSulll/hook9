Работа с запасами

Шаг 1. Войдите в раздел "Запасы".

Шаг 2. Выберите интересующий вас продукт из списка.

Шаг 3. Измените данные о текущем количестве или добавьте новую запись о поступлении.

Шаг 4. Сохраните изменения, нажав кнопку "Сохранить".

Контроль качества

Шаг 1. Перейдите в раздел "Контроль качества".

Шаг 2. Введите данные лабораторного анализа.

Шаг 3. Проверьте соответствие введенных данных предельным нормам.

Управление движением

Шаг 1. Войдите в раздел "Перемещение".

Шаг 2. Укажите тип движения (приход, расход, списание).

Шаг 3. Заполните связанные данные (объем, дату, причину).

Шаг 4. Сохраните запись.

Создание отчетов

Шаг 1. Перейдите в раздел "Отчеты".

Шаг 2. Укажите параметры отчета (период, тип данных).

Шаг 3. Нажмите "Создать отчет".

Шаг 4. Скачайте отчет в формате PDF или Excel.
